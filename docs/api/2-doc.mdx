# doc 配置

## 什么是 docz?
docz 是一个基于 mdx 的文档工具, 我们的文档功能基于其开发, 详情可见 [docz](https://github.com/pedronauck/docz)

## 示例
```js
export default {
  plugins: [
    ['umi-plugin-library', {
        /** doc 配置 */
        doc: {
            title: 'Umi 组件开发文档',
            theme: 'docz-theme-default'
        },
    }],
  ],
}
```

## title
* Type: `string`
* Default: `''`

标题

## theme
* Type: `string`
* Default: `'docz-theme-default'`

主题, 另外还有一个 [docz-theme-antd](https://clock157.github.io/umi-plugin-library-antd-demo/) 不成熟, 建设中.
