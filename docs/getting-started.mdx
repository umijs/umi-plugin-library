# 快速开始
> 如果你是内网用户, 请使用 [bigfish 脚手架](https://bigfish.alipay.com/) 

## 脚手架
> 脚手架提供一个最小的可运行实例

### 安装
```bash
# 创建目录
$ mkdir my-lib && cd my-lib

# 初始化脚手架, 选择 library
$ yarn create umi

# 安装依赖
$ yarn install
```

### 目录结构
```bash
├── .gitignore
├── .umirc.js       # umi 配置文件
├── LICENSE
├── README.md
├── package.json
└── src
    ├── index.css
    ├── index.js    # 组件
    └── index.mdx   # 文档
```

### 运行
```bash
$ yarn run dev
```

## 在现有项目中集成

### 安装插件

```bash
yarn add umi-plugin-library --save-dev
```

### 修改配置 .umirc.js
```js
export default {
  plugins: [
    'umi-plugin-library',
  ],
}
```

### 添加对应的 scripts
```json
"scripts": {
  "dev": "umi doc dev",
  "build": "umi lib build"
}
```

### 创建一个 hello.mdx
```mdx
---
name: Hello World
route: /hello
---

# Hello World
```

### 运行
```bash
$ yarn run dev
```
